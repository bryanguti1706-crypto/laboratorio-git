📚 Tutorial Básico de Git

Aprende Git desde cero de forma sencilla

¿Qué es Git?

Git es un sistema de control de versiones que permite guardar y gestionar cambios en proyectos, especialmente en programación.

¿Para qué sirve?

  • Guardar versiones de tu proyecto
  • Trabajar en equipo sin conflictos
  • Recuperar versiones anteriores

Comandos básicos

git init

Inicia un repositorio en tu proyecto.

git add .

Agrega todos los archivos para ser guardados.

git commit -m "mensaje"

Guarda los cambios con una descripción.

git push

Sube los cambios al repositorio remoto.

Consejo 💡

Haz commits frecuentes y con mensajes claros para mantener tu proyecto ordenado.
